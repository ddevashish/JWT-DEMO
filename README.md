## Developers

If you want to run the latest code from git, here's how to get started:

1. Clone the code:

    git clone https://github.com/ddevashish/JWT-DEMO.git

2. Install the dependencies

    npm install

3. Build the code

    npm run build

4. Run

    npm start
    or
    npm dev
